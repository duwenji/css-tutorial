::first-line

この段落の最初の行だけが太字で大きくなります。 ブラウザの幅を変えると、折り返し位置によって 最初の行の範囲が変わることが確認できます。 2行目以降は通常のスタイルで表示されます。

::first-letter（ドロップキャップ）

こちらの段落は最初の文字だけが大きく赤色で表示されます。 雑誌やニュースサイトでよく見られるドロップキャップの 表現に使われるテクニックです。
