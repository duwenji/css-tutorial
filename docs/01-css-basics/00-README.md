# 01. CSS Basics - 基礎知識

このカテゴリでは、CSSの基礎文法と概念を体系的に学びます。
Layout Design Guidelines を理解するために必要な土台をここで固めます。

## 学習目標

- CSSセレクタと詳細度の仕組みを理解する
- ボックスモデルを正確に把握する
- カスケード・継承の動作を理解する
- displayプロパティの種類と特徴を知る
- positionプロパティによる配置を習得する

## 教材一覧

| # | 教材 | 内容 |
|---|------|------|
| 01 | [セレクタと詳細度](01-selectors-and-specificity.md) | 要素選択の仕組みと優先順位 |
| 02 | [ボックスモデル](02-box-model.md) | margin/border/padding/contentの構造 |
| 03 | [カスケードと継承](03-cascade-and-inheritance.md) | スタイル適用の優先ルール |
| 04 | [displayプロパティ](04-display-property.md) | block/inline/flex/gridの使い分け |
| 05 | [position指定](05-positioning.md) | static/relative/absolute/fixed/sticky |

## 学習の進め方

01 → 05 の順に進めることを推奨します。
各教材の「理解度チェック」で習得状況を確認してください。
