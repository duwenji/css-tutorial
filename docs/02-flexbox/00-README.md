# 02. Flexbox - 1次元レイアウト

このカテゴリでは、Flexboxの全プロパティと実践的な使い方を学びます。
Flexboxはレイアウト設計原則の核心技術です。

## 学習目標

- フレックスコンテナの全プロパティを理解する
- フレックスアイテムの全プロパティを理解する
- `flex` ショートハンドを正確に使いこなせる
- 演習課題で実践的なレイアウトを組める

## 教材一覧

| # | 教材 | 内容 |
|---|------|------|
| 01 | [コンテナプロパティ](01-flex-container.md) | display/flex-direction/justify-content/align-items/gap/flex-wrap |
| 02 | [アイテムプロパティ](02-flex-items.md) | flex/flex-grow/flex-shrink/flex-basis/align-self/order |
| 03 | [演習課題](03-flexbox-exercises.md) | 実践的なレイアウト構築 |

## 学習の進め方

01 → 02 → 03 の順に進めてください。
特に `min-height: 0` の重要性はレイアウト設計原則に直結する重要ポイントです。
