# 画面イメージ画像

このディレクトリには、各教材の「実ソースコード」をブラウザで表示した
画面イメージ（スクリーンショット）を配置します。

## 画像キャプチャ手順

1. 各教材の「実ソースコード」をコピーして `.html` ファイルとして保存
2. ブラウザで開く
3. ブラウザの開発者ツールで `600px` `768px` `900px` `1080px` の各高さを確認
4. デフォルトの高さ `900px` でスクリーンショットを撮影
5. PNG形式で保存し、このディレクトリに配置

## ファイル命名規則

```
{カテゴリ番号}-{教材番号}-{教材名}.png
```

例:
- `01-01-selectors-and-specificity.png`
- `02-01-flex-container.png`
- `04-01-height-propagation.png`
- `05-03-admin-panel.png`

## 注意事項

- 画像は電子書籍（PDF/EPUB）に含めるため、幅 **1200px** 以下にリサイズ推奨
- ファイルサイズは **1ファイル 200KB 以内** を目安に
- ブラウザは Chrome または Edge を使用し、OS標準のスクリーンショット機能でキャプチャ
