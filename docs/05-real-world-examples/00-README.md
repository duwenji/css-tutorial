# 05. Real World Examples - 実践演習

このカテゴリでは、レイアウト設計原則を活用した実践的な画面レイアウトを構築します。
これまでの全知識を統合し、実際のアプリケーション開発で通用するレイアウト設計力を身につけます。

## 学習目標

- レイアウト設計原則に完全準拠したダッシュボードレイアウトを構築できる
- チャットUIのような複雑なスクロール要件を実装できる
- 管理画面の多機能レイアウトを設計できる

## 教材一覧

| # | 教材 | 内容 | 設計原則 焦点 |
|---|------|------|----------------|
| 01 | [ダッシュボード](01-dashboard-layout.md) | サイドバー＋メインの基本構成 | 高さ伝播 / スクロール戦略 |
| 02 | [チャットUI](02-chat-layout.md) | メッセージリスト＋入力欄 | 複数スクロール境界 / flex: 1 |
| 03 | [管理画面](03-admin-panel.md) | テーブル＋フォーム＋パネル | Grid + Flex 複合 / チェックリスト |

## 共通の設計方針

各演習は以下のレイアウト設計原則に従います：

1. **固定値は html/body/#root のみ**
2. **高さ伝播は flex: 1 + min-height: 0**
3. **overflow は末端のみ**
4. **7項目チェックリストで検証**
