# 03. CSS Grid - 2次元レイアウト

このカテゴリでは、CSS Gridの全プロパティと実践的な使い方を学びます。
Gridは行と列を同時に制御する2次元レイアウトに最適です。

## 学習目標

- グリッドコンテナの全プロパティを理解する
- グリッドアイテムの全プロパティを理解する
- `fr` 単位を使いこなせる
- GridとFlexboxの使い分けを判断できる

## 学習の進め方

01 → 02 → 03 の順に進めてください。
GridはFlexboxと補完関係にあり、両方を使いこなすことで柔軟なレイアウトが可能になります。

## Grid vs Flexbox

| 項目 | Flexbox | CSS Grid |
|------|---------|----------|
| 次元 | 1次元（行または列） | 2次元（行と列） |
| 制御方向 | 主軸方向 | 両方向同時 |
| 用途 | コンポーネント内部の配置 | ページ全体の大枠レイアウト |
| 折り返し | flex-wrap | grid-templateで定義 |
