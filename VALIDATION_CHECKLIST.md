# バリデーションチェックリスト

## 教材品質チェック

各教材ファイルが以下の基準を満たしているか確認します。

### 体系化
- [ ] 学習目標が明記されている
- [ ] 完了条件が明確である

### 簡潔
- [ ] 重複説明がない
- [ ] 冗長な全文転載がない

### 読みやすさ
- [ ] 見出し順が統一されている
- [ ] 用語が一貫している

### 実用性
- [ ] 最小実行手順がある
- [ ] コピペ可能なコード例がある

### 整合性
- [ ] 説明とサンプルコードが矛盾しない
- [ ] リンク切れがない

### CSS Layout Guidelines 連携
- [ ] `css-layout-guidelines.md` の原則が適切に参照されている
